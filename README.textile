h2. Installation

To install *eport* gem just do this:

  gem install eport

  
h2. Using inside Rails

To use *eport* gem inside Rails you must have @eport/config.yml@ file, which must looks like this:

pre.
  :eport:
    :point: "your point number"
    :private_key_path: "path relative RAILS_ROOT"
    :host: "e-port host"
  
Then, inside your Rails app you can use eport for making several types of operations:

eport = Eport.new
eport.refill_operation(operation_name, operation_id, product_id, value, account)


h2. Supported operations

You can do the following operations:

* CHECK
* OPERATION
* CONFIRM
* CANCEL


h2. Additional functionality

Fetching catalog of *eport* service providers:

eport = Eport.new
eport.get_catalog


Fetching *eport* balance

eport = Eport.new
eport.get_balance
